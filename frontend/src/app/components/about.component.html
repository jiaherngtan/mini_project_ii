Welcome! This project uses the TMDB API.

This application is for all the movie lovers like me. Its purpose is to create a comprehensive database of movies that spans across genres, languages, and countries. With these information, maybe you can make informed decisions about what to watch.

Personally, I love to record down the movies that I want to watch and movies that I have already watched somewhere. For those movies that I have watched, I would like to leave a rating and review to them as well. This application is able to perform all of these.

Feel free to explore the application and I am always happy to help with any questions or feedback you may have, so that I can continue to work on to improve the application and add new features if possible. Thank you.
